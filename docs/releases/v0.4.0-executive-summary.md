## 1. Visão Geral

A **v0.4.0 — Institutional Layer Complete** representa o encerramento oficial de toda a **camada institucional** da Terus Platform — a primeira versão completa e navegável da plataforma como produto de **Supply Chain Intelligence** voltado ao mercado enterprise.

Esta versão consolida identidade visual, narrativa comercial, prova social, conversão e documentação técnica em uma **baseline permanente**, publicada em produção e pronta para apresentação a investidores, clientes e parceiros.

A partir deste marco, o frontend institucional entra em **modo congelado**. O próximo ciclo oficial do projeto é a **Sprint 1 — Backend Foundation**, que iniciará a construção do produto (API, autenticação, banco de dados e integração real com o frontend).

## 8. Próximo Objetivo

### Sprint 1 — Backend Foundation

Estabelecer a fundação técnica do backend e da autenticação, preparando o monorepo para onboarding funcional sem alterar a baseline institucional congelada.

| Entrega prevista | Tecnologia / escopo |
|------------------|---------------------|
| API estruturada | **FastAPI** async · camadas Router → Service → Use Case → Repository |
| Autenticação | JWT RS256 (access 15 min) + refresh HttpOnly cookie (7 dias) |
| Banco de dados | **PostgreSQL 16** · schemas `public`, `onboarding`, `audit` · Alembic |
| Route groups frontend | `(onboarding)` e `(admin)` — layouts e guards |
| Integração frontend | **TanStack Query** · primeiras APIs (`/health`, `/v1/auth/*`) |
| Design System | Expansão de `@terus/ui` conforme wireframes |

**Status:** planejado · documentado · **não autorizado para implementação imediata**.

Baseline completa: [`docs/backend-foundation.md`](../backend-foundation.md)

## 9. Considerações Finais

A **v0.4.0 — Institutional Layer Complete** é a **primeira versão institucional completa** da Terus Platform. Ela transforma o projeto de um repositório em construção para um **produto apresentável**, com identidade enterprise, narrativa comercial coerente, prova social validada e registro histórico permanente.

O site em produção demonstra, de forma autônoma, a proposta de valor da Terus: **Supply Chain Intelligence** — inteligência operacional que age, não apenas informa. Cases reais, métricas da Rede Terus e a jornada dos seis módulos comprovam credibilidade antes de qualquer integração técnica com ERP.

A partir deste ponto, o foco do projeto muda de **construção institucional** para **construção de produto**. A baseline v0.4.0 permanece intacta como referência; toda evolução futura parte dela, não a substitui.
