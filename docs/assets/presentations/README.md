# Apresentações — Terus Platform

> Repositório oficial de decks, slides e materiais de apresentação utilizados em reuniões executivas, comerciais e com investidores.

---

## Estrutura por versão

```
docs/assets/presentations/
├── README.md           ← este arquivo
├── v0.4.0/             ← apresentações da baseline institucional
├── v0.5.0/             ← futuras versões
└── ...
```

Cada pasta `v{version}/` armazena os materiais utilizados **durante o ciclo de vida** daquela versão.

---

## O que arquivar

| Tipo | Exemplo | Formato |
|------|---------|---------|
| Pitch executivo | Apresentação para CEO / investidor | PDF + PPTX |
| Demo comercial | Deck usado em reuniões de vendas | PDF |
| All-hands interno | Status da plataforma para equipe Terus | PDF |
| Workshop técnico | Arquitetura pré-Sprint 1 | PDF |

---

## Nomenclatura

```
{AAAA-MM-DD}-{audiencia}-{titulo}.{ext}
```

Exemplos:

- `2026-07-01-executivo-terus-platform-v0.4.0.pdf`
- `2026-07-15-comercial-demo-supply-chain-intelligence.pptx`
- `2026-08-01-investidor-roadmap-pos-institutional.pdf`
